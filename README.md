## 💻 소개
오징어 게임을 컨셉으로 디자인 한 웹 게임 사이트

퍼즐, 업 다운, 묵찌빠 게임을 하고 랭킹을 기록해보세요.

홈 화면 추가로 모바일에서도 편하게 접속 가능!
